# PDFMerge

> Modern PDF toolkit — merge, split, rotate, and rearrange pages right in your browser.

This repository contains only the built distribution files. The source code is not publicly available.

## Features

- Merge multiple PDF files into a single document
- Split PDFs by page range or extract individual pages
- Rotate pages in 90-degree increments
- Drag-and-drop page thumbnails to reorder them
- Remove or duplicate specific pages
- Compress PDF file size during export
- Add page numbers and watermarks
- Preview page thumbnails before applying changes
- Download the resulting PDF instantly

## Why PDFMerge?

No files are ever uploaded to a server. PDFMerge uses PDF.js and client-side JavaScript to process everything locally. Your documents stay private and secure. There are no daily limits, no sign-ups, and no costs — just a fast, reliable PDF tool that works offline.
